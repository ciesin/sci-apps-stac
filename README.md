# CIESIN GRID3 STAC

## STAC Catalog for CIESIN GRID3 data
This repository contains a STAC Catalog for the CIESIN GRID3 data. The catalog is organized into collections based on the different datasets available at CIESIN and in the GRID3 project. Each collection contains items that represent individual data files, along with metadata describing the contents of those files.
